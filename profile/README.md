## 💡 Overview

<a href="#firestormCAD">FirestormCAD</a> is a professional computer-aided design application for Mac delivering precise 2D drafting and 3D modeling capabilities for engineers, architects, and technical designers who require accuracy and professional output in their design workflows.

The <a href="#firestormCAD">parametric constraint system</a> maintains geometric relationships between design elements — defining that two lines must remain parallel, that a circle must remain tangent to a line, or that a dimension must hold a specific value regardless of how surrounding geometry changes. These constraints let designers modify one part of a design with confidence that related geometry updates correctly rather than manually adjusting every dependent element. The <a href="#firestormCAD">2D drafting toolkit</a> provides the complete set of technical drawing tools — lines, arcs, circles, polylines, splines, ellipses, and complex geometric construction tools alongside precision snap modes, grid alignment, and coordinate entry for accurate placement of every drawing element.

The <a href="#firestormCAD">layer management system</a> organizes drawing elements into named, color-coded layers — separating structural elements from dimensions, annotations from construction lines, and different design disciplines from each other in a single drawing file. Layer visibility and locking controls manage the complexity of technical drawings with many overlapping element types. The <a href="#firestormCAD">DXF and DWG compatibility</a> opens and saves the industry-standard AutoCAD file formats — exchanging drawings with collaborators, clients, and downstream workflows regardless of which CAD application they use, maintaining the interoperability that professional technical design requires.</p>

<a href="#firestormCAD">Dimensioning and annotation tools</a> add the measurements, notes, and callouts that transform geometric drawings into technical documents — linear dimensions, angular dimensions, radius and diameter callouts, leaders, text annotations, and tolerance specifications that communicate design intent to manufacturers and fabricators. <a href="#firestormCAD">Snap and object tracking</a> captures precise geometric relationships during drawing — snapping to endpoints, midpoints, intersections, perpendicular points, and tangent points for placement accuracy that manual coordinate entry would require more effort to achieve.

<a href="#firestormCAD">Print and export workflows</a> produce drawing outputs for professional use — scaled paper prints, PDF export for distribution, and raster image export for presentation materials. <a href="#firestormCAD">Custom tool palettes and shortcuts</a> configure the FirestormCAD interface around specific workflow requirements, placing the most-used tools at immediate access and mapping keyboard shortcuts to the operations that repeat most frequently in a given design context.</p>

## 💬 FAQ

<details>
<summary>Does FirestormCAD open AutoCAD DWG files?</summary>

Yes. FirestormCAD opens and saves DXF and DWG files for exchanging drawings with AutoCAD and AutoCAD-based applications.

</details>

<details>
<summary>What is the parametric constraint system?</summary>

Parametric constraints maintain geometric relationships between drawing elements — parallel lines stay parallel, tangent curves maintain tangency, dimensions hold specified values — when other parts of the design change.

</details>

<details>
<summary>Does FirestormCAD support 3D modeling?</summary>

Yes. 3D modeling capabilities include extruding 2D profiles into 3D solids, boolean operations between bodies, and 3D viewing for design verification.

</details>

<details>
<summary>How does the layer system work?</summary>

Layers are named, color-coded organizational containers for drawing elements. Each element belongs to a layer, and layers can be made visible, hidden, or locked independently.

</details>

<details>
<summary>Does FirestormCAD run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>

<details>
<summary>Can FirestormCAD export to PDF?</summary>

Yes. PDF export produces scaled technical drawings for distribution and review.

</details>

<details>
<summary>Does FirestormCAD support dimensioning?</summary>

Yes. Linear, angular, radius, diameter dimensions, leaders, and text annotations are available for complete drawing documentation.

</details>

<details>
<summary>What snap modes are available?</summary>

Endpoint, midpoint, intersection, perpendicular, tangent, and other geometric snap modes capture precise positions during drawing.

</details>

<details>
<summary>Can I create reusable symbols in FirestormCAD?</summary>

Yes. The block and symbol library stores reusable drawing elements for insertion without redrawing.

</details>
